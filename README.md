# Online Book Catalogue

## Project Overview
This project is a simple **web-based catalogue page** that displays a list of books.

---

## Functional Requirements

1. The system shall display a catalogue of books.
2. Each book shall include the following information:
   - Cover image
   - Title
   - Author
   - Category
   - Price
3. The system shall allow users to search books by:
   - Title
   - Author
4. The system shall allow users to filter books by category.
5. The catalogue shall update dynamically without page reload.
6. The system shall display a message when no books match the selected filters.

---

## Non-Functional Requirements

1. Usability
   - The interface shall be intuitive and easy to use.
   - Controls (search and filter) shall be clearly visible.

2. Performance
   - Filtering and searching shall be performed on the client side.
   - Page updates shall occur instantly without reloading.

3. Responsiveness
   - The page layout shall adapt to different screen sizes (desktop and mobile).

4. Maintainability
   - HTML shall be semantic and use meaningful class names.
   - CSS shall be stored in an external file.
   - JavaScript logic shall be separated from HTML.

5. Accessibility
   - Images shall include alternative text.
   - Semantic HTML elements shall be used (`header`, `main`, `section`, `article`, `footer`).
